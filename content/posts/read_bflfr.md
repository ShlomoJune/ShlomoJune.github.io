## 他人批注



{{< admonition type=note title="Note" open=true >}}
一味听父母的话只能成为他们的样子，按照自己的想法勇敢尝试才能活出自己。
{{< /admonition >}}

{{< admonition type=note title="Note" open=true >}}
一个没有自我的女人，当她的伴侣在爱上有缺失时，她便很容易将这份期待寄托于孩子身上，孩子就是生命的全部。而当孩子长大后独立时，她的寄托就散架了。
{{< /admonition >}}

{{< admonition type=note title="Note" open=true >}}
所以你一旦看到哪个女人把小孩当成为生命的全部时，就会猜到她的另一半可能在爱的输送方面一定是有缺失的，这就是所谓的情感转移。然而，一旦小孩长大后更独立地面对自己的生活，母亲的失落感将会成倍的增长，会发现多年来唯一的情感纽带越来越薄弱，可怜又可悲！
{{< /admonition >}}

{{< admonition type=note title="Note" open=true >}}
“每当你想批评人的时候，要记得，世上不是所有人都像你一样拥有许多优势。”
——《了不起的盖茨比》
{{< /admonition >}}

{{< admonition type=note title="Note" open=true >}}
夸聪明是错误的教育方式，因为夸聪明会让孩子觉得，事情取得成功是因为他们足够聪明。可是当遇到难题无法一下解决的时候，他们就会认为是自己不够聪明然后早早放弃。相反，那些被夸努力的人，做事情会更坚韧更有毅力，这样的人才能取得大的成就。另一方面，认为自己聪明的人，总想着自己比别人效率高然后时不时偷懒，不肯踏踏实实。
{{< /admonition >}}

{{< admonition type=note title="Note" open=true >}}
人能存于世，很大程度上认为自己有价值，一直在寻找价值认同感，如今这个世界上少了一个认可自己一定价值的人，一定会有缺失感。
{{< /admonition >}}

{{< admonition type=note title="Note" open=true >}}
世界上有两种悲剧，一种是求而不得，一种是得偿所愿。—王尔德
{{< /admonition >}}

{{< admonition type=note title="Note" open=true >}}
真正注重上流社会规则的往往不是上流社会的人
{{< /admonition >}}

{{< admonition type=note title="Note" open=true >}}
人群聚集的地方，无非就三件事，拐弯抹角的炫耀自己，添油加醋的贬低别人，相互窥探搬弄是非
{{< /admonition >}}

{{< admonition type=note title="Note" open=true >}}
用宏大的世界稀释痛苦，用微小的事情感知幸福。
{{< /admonition >}}

{{< admonition type=note title="Note" open=true >}}
“再干净的人,也会被指点。人们不是因为我们有了过错才指点,而是指点了我们,才能显得他们是干净的人。”——«观鹤笔记»
{{< /admonition >}}

{{< admonition type=note title="Note" open=true >}}
情绪不断滋长，没有宣泄的出口，就会肆意蔓延至泛滥，吞噬理智。

{{< admonition type=note title="Note" open=true >}}
劳动可以对抗虚无，但前提是这种劳动与人的内在需求产生共鸣。
{{< /admonition >}}

我希望我写的这段话有人可以反驳我：
没有人会透过你的外表去看你的灵魂。
所谓爱情就是见色起意，婚姻更是各有图谋、一场交易。
真实生活中没有简爱这样的故事，丑女孩不会因为善良坚强被爱，如果真正被看到了金子般的心，也只会被尊重，但那也不是爱。
无论多么伟大的人，在选择爱情时都是残忍而又肤浅的。

在没有建立成熟的人生观世界观之前，所有浮于表面的艺术形式和作品有时候会让人迷失

你现在碰到的每个困境，基本都是针对你性格的弱点而量身完制的，所以命运反复出题，直到你给出新的答案，直到你跨过去。
